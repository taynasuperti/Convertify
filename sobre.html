Sobre o Convertify

O Coinvertify é um conversor de moedas simples, prático e totalmente gratuito. Desenvolvido como um projeto front-end, ele utiliza HTML, CSS e JavaScript para oferecer uma experiência leve e intuitiva, com conversões automáticas e em tempo real graças a uma API de câmbio confiável.

O objetivo do Coinvertify é facilitar a vida de quem precisa fazer conversões rápidas entre moedas do mundo todo — seja para viagens, compras online ou estudos. Com uma interface amigável e responsiva, o usuário pode selecionar as moedas de origem e destino e obter o valor convertido com apenas alguns cliques.

Este projeto foi criado com foco em aprendizado e prática de integração com APIs, manipulação do DOM e boas práticas de design responsivo. Além disso, busca apresentar uma solução visualmente agradável, com navegação fluida e organização de código clara.

Funcionalidades Principais:

Conversão em tempo real entre diversas moedas;

Interface leve e responsiva;

Atualização automática das taxas de câmbio;

Visual moderno e fácil de usar.

Tecnologias Usadas:

HTML5 e CSS3 para estrutura e estilo;

JavaScript para lógica e integração com a API;

API de câmbio gratuita.

Aprendizados adquiridos com o projeto:

Integração com APIs REST;

Tratamento de dados assíncronos;

Criação de uma interface limpa e funcional;

Organização e clareza no código.
